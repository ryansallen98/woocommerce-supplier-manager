*** Changelog ***

= 1.3.1 - 2025-09-10 =
* Fix - Fix bug where Supplier Order columns and filters where not being controlled via admin and changes were not saved. 

= 1.3.0 - 2025-09-10 =
* New - Ability to toggle what orders are visible, filterable and sortable for the Supplier Orders table via order status.  
* Tweak - Suppliers get an email notification when an order status changes to refunded or cancelled.
* Tweak - Restricted supplier orders to only show and filter confirmed statuses.
* Tweak - Supplier name in all selects/dropdowns in the WP Admin now reflect the supplier display name chosen by the admin.
* New - Add general settings tab/page with settings to change global supplier names; Company Name, Full Name, Wordpress Display Name...
* Tweak - Move supplier order column controls to Orders Dashboard tab.
* New - Add tabs to supplier settings for more options and better navigation.
* New - Add company information fields to users with the supplier role.

= 1.2.1 - 2025-09-10 =
* Fix - Fix crashing bug on supplier products dashboard page

= 1.2.0 - 2025-09-10 =
* New - Filter by supplier in WP Admin products table
* New - Supplier Column in WP Admin products table

= 1.1.0 - 2025-09-09 =
* Fix - Fixed bug where supplier price input was not rendering under product variations.
* New - Added **Supplier Settings** page under WooCommerce admin.  
* New - Ability to toggle **column visibility, sorting, and filter enable/disable** for the Supplier Orders table.  
* New - Added **Help tab** and inline **accordion help box** with developer instructions and code examples.  
* New - Plugin **Settings link** now appears directly on the Plugins page.  
* Tweak - All columns (even hidden ones) are always available in the settings page, so they can be re-enabled.  
* Tweak - Filters are now toggleable (on/off) without changing their type.  
* Dev - Improved `wcsm_supplier_orders_columns` API to support 3rd-party columns with settings integration.  

= 1.0.1 - 2025-09-09 =
* Fix - Fixed crashing bug with auto updater.  

= 1.0.0 - 2025-09-09 =
* First release.
